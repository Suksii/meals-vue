# Meals app (vue)

## About
Project developed in Vue. In this app you can search for meals by name, first letter, category

## Technical Details
- **Vue.js:**
  - `ref, reactive, watch, onMounted, computed`
  - `axios`
  - `vuex`
  - `vue-router`
  - `swiper`
  - `tailwindcss`
 
- **JavaScript:**
  - `trim, push`
